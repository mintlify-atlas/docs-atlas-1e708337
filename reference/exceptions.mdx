Todas las excepciones del sistema extienden directamente `java.lang.Exception` (excepciones verificadas). Cada una se lanza cuando una regla de negocio o de validación no se cumple, y su mensaje describe la causa.

**Paquete:** `com.nousacademy.excepciones`  
**Total:** 37 clases

## Validación de campos

Excepciones lanzadas durante la creación o actualización de entidades cuando los datos de entrada no cumplen los requisitos mínimos.

<AccordionGroup>

<Accordion title="ElementosVacios">

**Mensaje:** `"Campos vacios no permitidos"`

Lanzada cuando uno o más campos obligatorios llegan vacíos (`isEmpty()`) o nulos al constructor o a un método de negocio.

```java
public class ElementosVacios extends Exception {
    public ElementosVacios() {
        super("Campos vacios no permitidos");
    }
}
```

**Usada en:** constructores de `Usuario`, `Estudiante`, `Docente`, `Clase`, `Grupo`; métodos `asignarMembresiaPersonalizada`, `actualizarEstadoContrato`, entre otros.

</Accordion>

<Accordion title="NombreNoValido">

**Mensaje:** `"Nombre demasiado corto"`

Lanzada cuando `nombre` o `apellidos` tienen menos de 2 caracteres.

```java
public class NombreNoValido extends Exception {
    public NombreNoValido() {
        super("Nombre demasiado corto");
    }
}
```

**Usada en:** `Usuario` (constructor y `setNombre`, `setApellidos`).

</Accordion>

<Accordion title="CorreoNoValido">

**Mensaje:** `"Los correo deben pertenecer a la organizacion - terminar en: @nousacademy.co"`

Lanzada cuando el correo no contiene `@` o no termina en `nousacademy.co`.

```java
public class CorreoNoValido extends Exception {
    public CorreoNoValido() {
        super("Los correo deben pertenecer a la organizacion - terminar en: @nousacademy.co");
    }
}
```

**Usada en:** `Usuario.setCorreo`, `AcademiaController.setCorreo`.

</Accordion>

<Accordion title="LongitudNoValida">

**Mensaje:** `"Campos con caracteres no suficientes"`

Lanzada cuando `telefono` o `numeroDocumento` tienen menos de 5 caracteres.

```java
public class LongitudNoValida extends Exception {
    public LongitudNoValida() {
        super("Campos con caracteres no suficientes");
    }
}
```

**Usada en:** constructor de `Usuario`.

</Accordion>

<Accordion title="ValoresNegativos">

**Mensaje:** `"Campos con valores negarivos no permitidos"`

Lanzada cuando un campo numérico recibe un valor negativo donde no está permitido (teléfono, número de documento, precio, salario, duración, aforo, etc.).

```java
public class ValoresNegativos extends Exception {
    public ValoresNegativos() {
        super("Campos con valores negarivos no permitidos");
    }
}
```

**Usada en:** `Usuario`, `Membresia`, `Contrato`, `Reserva`, `AcademiaController.setTelefono`, `inyectarCapital`, `retirarCapital`, entre otros.

</Accordion>

<Accordion title="FechaInvalida">

**Mensaje:** `"Fecha no valida, es anterior a la actual"`

Lanzada cuando una fecha de inicio (`LocalDate` o `LocalDateTime`) es anterior al momento actual.

```java
public class FechaInvalida extends Exception {
    public FechaInvalida() {
        super("Fecha no valida, es anterior a la actual");
    }
}
```

**Usada en:** constructores de `Membresia`, `Contrato`; constructor de `Reserva`.

</Accordion>

<Accordion title="CalificacionInvalida">

**Mensaje:** `"La calificacion solo puede darse entre 1 y 5"`

Lanzada cuando el valor de calificación está fuera del rango `[1, 5]`.

```java
public class CalificacionInvalida extends Exception {
    public CalificacionInvalida() {
        super("La calificacion solo puede darse entre 1 y 5");
    }
}
```

**Usada en:** `Reserva.calificarClase`, `AcademiaController.calificarReserva`.

</Accordion>

</AccordionGroup>

## Búsqueda y duplicados

<AccordionGroup>

<Accordion title="ElementoNoExistente">

**Mensaje:** `"Elemento no encotrado"`

Lanzada cuando se busca una entidad (usuario, estudiante, docente, grupo o reserva) y no se encuentra ninguna coincidencia.

```java
public class ElementoNoExistente extends Exception {
    public ElementoNoExistente() {
        super("Elemento no encotrado");
    }
}
```

**Usada en:** todos los métodos `buscarUn*` del controlador; `eliminarUsuario`, `eliminarEstudiante`, `eliminarDocente`, `eliminarGrupo`, `eliminarReserva`.

</Accordion>

<Accordion title="Duplicados">

**Mensaje:** `"Datos ya existentes"`

Lanzada al intentar crear una entidad cuyo número de documento o nombre de usuario ya existe en el sistema; también al intentar añadir un estudiante que ya está en un grupo.

```java
public class Duplicados extends Exception {
    public Duplicados() {
        super("Datos ya existentes");
    }
}
```

**Usada en:** `crearUsuario`, `crearEstudiante`, `crearDocente`, `Grupo.anadirEstudiante`.

</Accordion>

</AccordionGroup>
